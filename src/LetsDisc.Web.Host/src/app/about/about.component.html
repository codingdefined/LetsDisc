About LetsDisc

LetsDiscuss is a place where programmers around the world can share ideas and help each other grow by sharing knowledg, experience and wisdom.

We can think of it as an online community for sharing and discovering great ideas, great questions and great answers. Any registered user can ask questions, can also answer questions asked by other users

This website has been founded by Gopesh Sharma, Full Stack Developer and the code being open source can be found at Github.
